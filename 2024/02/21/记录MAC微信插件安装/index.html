安装微信助手（mac版）

之前在网络上看到过别人的定制版微信，所以搜索了一下，已知的一些用于iphone的定制版微信都是需要收费的，而且也不太安全，所以退而求其次，想着安装一个mac版的微信插件。

此文章记录一下安装插件的过程以及遇到的一些问题和解决方式，总体来说安装还是很简单的

github 仓库地址: https://github.com/MustangYM/WeChatExtension-ForMac
仓库中的介绍已经很详细了，包括安装、卸载、以及插件的功能介绍，这里只补充一些安装问题

安装方式

推荐懒癌版安装方式：使用 On My WeChat（我也是用的这种方式安装的，很润很顺滑）

打开应用程序-实用工具-Terminal(终端)，执行下面的命令安装 Oh My WeChat

命令使用方式

omw 会从 GitHub 仓库检查更新及下载安装包，但由于网络不稳定，下载可能会失败，但你还可以使用 omw load 命令安装小助手。

遇到的问题

一般来说使用以上命令后，微信助手就可以安装成功了，但是还是会有一些问题，以下就是我遇到的两个问题，在此记录一下

问题一：电脑上的微信版本太高，使用安装小助手安装插件后微信无法打开

解决方法：建议使用 3.8.2 版本的微信
安装地址：https://github.com/zsbai/wechat-versions/releases?page=1

问题issue：https://github.com/MustangYM/WeChatExtension-ForMac/issues/1835
